Documentation phase for Project 2.

Run main.py for the sample.

Running the program will display the contents of "sample.txt" via "txtprocessing.txtreader.py's" functionality, 
the total lines in "sample.txt" via "txtprocessing.txtanalyzer.py's" functionality, and finally it will convert
"csv_sample.txt" to "converted_text.csv" via "txtprocessing.txtconverter.py's" functionality.

Public Libraries/Modules used: Tkinter (GUI), pandas (text conversion)
